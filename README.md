# Ukraine-Euro Area Inflation Counterfactual

This repository contains the code, data, and outputs for my Quantitative Methods in Finance final exam project:

**Counterfactual Inflation Analysis: What If Ukraine Had Been Part of the Euro Area?**

## What this project does

The project builds a counterfactual inflation path for Ukraine under the hypothetical case where Ukraine had been part of the Euro Area.

The idea is to estimate bivariate structural VARs for Ukraine and the Euro Area using inflation and industrial production growth. Following the Blanchard-Quah long-run restriction used in the Bayoumi-Eichengreen logic, I recover structural demand shocks and replace part of Ukraine's demand shocks with Euro Area demand shocks.

The shock replacement is regime-based. It is stronger during the periods where monetary sovereignty mattered the most, especially devaluation and flexible-regime episodes. Peg and stabilized periods receive a lower treatment intensity, because Ukraine already had limited monetary autonomy in practice during those years.

Oil prices are included as an exogenous control.

## How to reproduce the results

First install the required packages:

```bash
pip install -r requirements.txt
```

Then run the notebooks in this order:

```text
code/01_exchange_rate_chronology.ipynb
code/02_svar_counterfactual.ipynb
```

The notebooks generate the cleaned datasets, final tables, and figures used in the report.

The main outputs are saved in:

```text
outputs/figures/
outputs/tables/
```

## Data

The project uses the exam-provided inflation datasets and external macroeconomic data needed for the identification strategy:

- Ukraine CPI data
- Euro Area HICP data
- Ukraine industrial production
- Euro Area industrial production
- USD/UAH exchange-rate data
- oil prices

The final report documents the sources and explains why each variable is used.
